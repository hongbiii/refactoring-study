## 진행방식

- 기본적으로 1주일에 **1챕터** 분량을 준비해온다.
    - 중요하다고 생각하는 부분을 **코드로 준비**해오기
    - 테스트 코드는 **4장 전까지 자유**

- 매주 **진행자 1명**이 아래 역할을 수행한다.
    - 진행자가 자신 주차의 챕터에 대한 의견을 제일 먼져 제시한다.
